### How to use

バイナリからPNGのシグネチャを解析し、PNG画像として抽出します。また、読み込んだバイナリデータを16進数で表示・編集することも可能です。

> [任意のバージョンでダウンロード](https://github.com/Sakamochanq/Image-Decoder/releases)

### Build

<br>

```bash
git clone https://github.com/Sakamochanq/Image-Decoder.git
cd Image-Decoder/src
```

<br>

Visual Studioから `*.sln` を開き Build を行ってください。
